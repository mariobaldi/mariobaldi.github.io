Algoritmi e Programmazione Avanzata (01EIP)

Materiale didattico

Materiale utilizzato nelle esercitazioni in aula

  • Lunedì 3 dicembre - tel.c: ricerca lineare in elenco telefonico letto da file (tel.txt)
  • Lunedì 10 dicembre
    • telBinIt.c: ricerca binaria realizzata in modo iterativo in elenco telefonico letto da file (tel.txt) e ordinato.
    • telBinRi.c: ricerca binaria realizzata in modo ricorsivo in elenco telefonico letto da file (tel.txt) e ordinato.
    • telMerge.c: ricerca binaria realizzata in modo ricorsivo in elenco telefonico letto da file (tel.txt) e ordinato utilizzando l'algoritmo merge sort.
  • Lunedì 17 dicembre
    • vettStr.c: ordinamento di un file di stringhe (strings.txt) usando un vettore di stringhe allocato dinamicamente.
    • vettPunt.c: ordinamento di un file di stringhe (strings.txt) usando un vettore di puntatori a stringhe allocato dinamicamente.
    • 8regine.c: risoluzione del problema del posizionamento di otto regine su una scacchiera realizzata in modo ricorsivo
  • Lunedì 7 gennaio
    • vita.c: gioco della vita
    • Lista: questo direttorio contiene i vari moduli per un programma che realizza inserimenti, ricerche e cancellazioni in una lista ordinata. Il file leggimi.txt contiene alcune informazioni per la compilazione.
  • Mercoledì 16 gennaio
    • Tema d'esame: Cammini distinti che collegano due nodi di un grafo (sorgenti)
    • Tema d'esame: Identificazione di itinerari di viaggio aereo (sorgenti)
  • Mercoledì 23 gennaio
    • Tema d'esame: assegnazione di aule a corsi in base ad una serie di richieste contenute in un file
    • Tema d'esame
      • Ricerca logaritmica in base dati secondo due chiavi. Il programma è organizzato in 2 moduli:
        • programma principale
        • dichiarazione dei tipi di dato e procedure operanti su di essi
      • Individuazione di sequenza ottima
